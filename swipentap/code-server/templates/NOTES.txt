code-server has been deployed.

1. Get the application URL by running:
{{- if .Values.ingress.enabled }}
  https://{{ (index .Values.ingress.hosts 0).host }}
{{- else }}
  export NODE_PORT=$(kubectl get --namespace {{ .Release.Namespace }} -o jsonpath="{.spec.ports[0].nodePort}" services {{ include "code-server.fullname" . }})
  export NODE_IP=$(kubectl get nodes --namespace {{ .Release.Namespace }} -o jsonpath="{.items[0].status.addresses[0].address}")
  echo http://$NODE_IP:$NODE_PORT
{{- end }}

2. The default password is stored in the secret {{ include "code-server.fullname" . }} (key: password).
   To retrieve: kubectl get secret {{ include "code-server.fullname" . }} -n {{ .Release.Namespace }} -o jsonpath='{.data.password}' | base64 -d

Continue + Ollama are preconfigured when continue.enabled is true (apiBase points at in-cluster Ollama).
